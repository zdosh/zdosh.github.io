# zds.github.io

Personal academic homepage template (prokia-style) for internship applications.

## Quick Edit Checklist

- Update name, school, and bio in `index.html`
- Replace links for GitHub / Scholar / CV / Email
- Replace `assets/profile.svg` with your real profile photo (keep the same filename or update the path)
- Add your real `cv.pdf` in repo root and keep the link `./cv.pdf`
- Replace placeholder project/publication entries with your own

## Deploy with GitHub Pages

1. Create repository: `<your-github-username>.github.io`
2. Push all files in this folder to `main`
3. Enable Pages from `main` branch root (if not auto-enabled)
4. Visit `https://<your-github-username>.github.io`
